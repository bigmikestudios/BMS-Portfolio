This plugin creates a content type and taxonomy for a portfolio section, as well as shortcodes for listing and display.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

This plugin depends upon the BMS Smart Meta Box plugin, it won\'t work unless it is also installed.
